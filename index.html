Indicadores

Dados Não Disponíveis na Fonte

O documento PDF "Estatística Municipal - Parauapebas" (FAPESPA) não fornece dados desagregados por gênero para os seguintes indicadores:

  • Mercado de Trabalho (taxa de ocupação, desemprego, renda média, diferença salarial).
  • Educação (nível de instrução, matrículas por nível, taxas de abandono/aprovação).

Portanto, não é possível preencher os gráficos desta seção com a fonte fornecida.

Sobre o Observatório

O que é o Observatório de Gênero?

O Observatório de Gênero do Município de Parauapebas é uma plataforma de monitoramento e análise de dados sobre a situação das mulheres no município.

Fontes de Dados

Os dados apresentados neste Observatório foram coletados das seguintes fontes principais:

  • Fundação Amazônia de Amparo a Estudos e Pesquisas (FAPESPA): "Estatística Municipal - Parauapebas" (2024).
  • Ministério da Saúde (SINAN): Notificações de violência interpessoal (2010-2024).
  • Dados secundários citados nos documentos: IBGE, DATASUS/MS, TRE, etc.
